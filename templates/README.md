## HTML:

### The required elements of a HTML document:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document Title</title>
  <link rel="stylesheet" href="main.css">
</head>
<body>
  Body of website
</body>
</html>
```
### Sections of an HTML document:
```html
<header></header>

<section>
  <article></article>
</section>

<aside></aside>

<footer></footer>
```
